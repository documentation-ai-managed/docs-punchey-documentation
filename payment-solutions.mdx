## Payment Processing Overview

Punchey provides comprehensive payment solutions designed for small businesses. Our platform supports multiple payment methods, ensures secure transactions, and offers detailed analytics to help you optimize your revenue streams.

<Steps>
  <Step title="Choose Payment Method" icon="credit-card">

  </Step>
  <Step title="Configure Settings" icon="settings">
    Set up transaction limits, fees, and notification preferences in your dashboard.
  </Step>
  <Step title="Test Transactions" icon="check-circle">
    Use our sandbox environment to test payment flows before going live.
  </Step>
</Steps>

<CodeGroup tabs="cURL,JavaScript">
  <Tab title="cURL" icon="terminal">
    ````bash
    curl -X POST https://api.punchey.com/v1/payments \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "amount": 5000,
        "currency": "USD",
        "payment_method": "card",
        "customer_id": "cust_123"
      }'
    ````
  </Tab>
  <Tab title="JavaScript" icon="code">
    ````javascript
    const response = await fetch('/api/payments', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${apiKey}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        amount: 5000,
        currency: 'USD',
        paymentMethod: 'card',
        customerId: 'cust_123'
      })
    });
    const payment = await response.json();
    ````
  </Tab>
</CodeGroup>
