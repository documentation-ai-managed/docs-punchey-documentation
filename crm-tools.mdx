<CodeGroup tabs="Python,Node.js">
  <Tab title="Python" icon="code">
    ````python
    import punchey
    
    # Initialize CRM client
    crm = punchey.CRM(api_key='your-key')
    
    # Create customer profile
    customer = crm.customers.create({
        'name': 'John Doe',
        'email': 'john@example.com',
        'phone': '+1234567890'
    })
    
    # Add transaction data
    crm.transactions.add(customer.id, {
        'amount': 150,
        'date': '2024-01-15',
        'items': ['widget-a', 'widget-b']
    })
    ````
  </Tab>
  <Tab title="Node.js" icon="code">
    ````javascript
    const punchey = require('punchey-sdk');
    
    // Initialize CRM client
    const crm = new punchey.CRM({ apiKey: 'your-key' });
    
    // Create customer profile
    const customer = await crm.customers.create({
      name: 'John Doe',
      email: 'john@example.com',
      phone: '+1234567890'
    });
    
    // Add transaction data
    await crm.transactions.add(customer.id, {
      amount: 150,
      date: '2024-01-15',
      items: ['widget-a', 'widget-b']
    });
    ````
